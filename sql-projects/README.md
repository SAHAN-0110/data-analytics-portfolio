# Retail Sales Analysis

## Project Overview
This project analyzes retail sales data to understand sales performance, customer behavior, and product trends.

The goal is to answer business questions using SQL queries and derive actionable insights.

---

## Dataset
The dataset contains transaction-level retail sales data including:
- Order ID
- Order Date
- Customer ID
- Product Category
- Quantity
- Price
- Total Sales

---

## Business Questions
- What is the total revenue generated?
- Which product categories perform best?
- What is the average order value?
- Which days or months have higher sales?
- Who are the top customers by revenue?

---

## Tools Used
- SQL

---

## Key Insights

- Electronics generated the highest revenue among all product categories.
- A small number of customers contributed a significant portion of total revenue.
- The average order value indicates moderate spending per transaction.
- Sales are spread across categories, showing diversified product demand.

These insights can help businesses focus on high-performing categories and valuable customers.
